USDA Loans (United States Department of Agriculture)

USDA is a program only available in low income median areas. Fortunately there is a interactive map that allows users to determine eligbility. You can also find income limits and other useful information.
    How to Use the Map:
  1. Open Link Below.
  2. Click "Single Family Housing Guaranteed"
  3. The site will prompt you to accept terms. Accept them.
  4. Use search bar on interactive map to search properties
  • USDA Elibility Map

THDA Loans (Tennessee Housing Development Agency)

THDA has no location restrictions but does have strict income limits among other rules. It's important for borrowers to know them. The links below provide sources on eligiblilty requirements, income restrictions, and required classes the borrower must take.

  • THDA Borrower Requirements
  • THDA Income Limits
  • Borrower Class Information

Conventional Homeready

Homeready is a newer program available for first time buyers. It only requires a down payment of 3% but has requirements including income limits and borrower classes.
